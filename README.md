# openQAclient

A Zig reimplementation of the `openqa-cli api` subcommand, distributed as
both a CLI executable and a static library.

---

## Prerequisites

- Zig 0.15.2

---

## Getting Started

```sh
zig build              # build library + executable
zig build test         # run all unit tests
```

See [CONTRIBUTING.md](CONTRIBUTING.md) for build targets, E2E / fuzz testing,
formatting, and release process.
